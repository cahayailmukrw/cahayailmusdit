# SDIT Cahaya Ilmu - Website Resmi

Website resmi untuk SDIT Cahaya Ilmu (Sekolah Dasar Islam Terpadu) yang dibangun dengan HTML, CSS, dan JavaScript modern.

## 📋 Fitur

- **Responsive Design** - Tampilan optimal di semua perangkat (desktop, tablet, mobile)
- **Navigasi Modern** - Menu navigasi yang user-friendly dengan smooth scrolling
- **Hero Section** - Landing page yang menarik dengan call-to-action
- **Profil Sekolah** - Informasi lengkap visi, misi, dan akreditasi
- **Program Unggulan** - Showcase program-program terbaik sekolah
- **Fasilitas Lengkap** - Galeri fasilitas sekolah yang modern
- **Statistik Dinamis** - Animasi counter untuk data sekolah
- **Form Pendaftaran** - Formulir pendaftaran online dengan validasi
- **Newsletter** - Sistem berlangganan informasi terbaru
- **Animasi Interaktif** - Smooth animations dan micro-interactions

## 🚀 Teknologi

- **HTML5** - Semantic markup yang modern
- **CSS3** - Styling dengan Flexbox dan Grid, animasi, dan transitions
- **JavaScript ES6+** - Modern JavaScript dengan best practices
- **Font Awesome** - Icons library
- **Google Fonts** - Typography (Poppins)

## 📝 Customization

### Mengubah Warna Tema
Edit CSS variables di `styles.css`:
```css
:root {
    --primary-color: #667eea;
    --secondary-color: #764ba2;
    --accent-color: #ffd700;
}
```

### Mengubah Font
Ganti font family di CSS:
```css
body {
    font-family: 'Your-Font', sans-serif;
}
```

### Menambahkan Section Baru
1. Tambahkan HTML section di `index.html`
2. Tambahkan styling di `styles.css`
3. Tambahkan JavaScript functionality jika diperlukan
