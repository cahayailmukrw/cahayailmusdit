# SDIT Cahaya Ilmu - Website Profil Sekolah

Website profil sekolah untuk SDIT Cahaya Ilmu yang menyajikan informasi umum tentang sekolah.

## 📋 Overview

Website profil sekolah dengan fitur:
- Responsive Design untuk semua perangkat
- Navigasi modern dengan smooth scrolling
- Hero section yang menarik
- Profil sekolah (visi, misi)
- Program unggulan
- Fasilitas lengkap
- Statistik sekolah
- Form kontak
- Newsletter subscription

## 🚀 Quick Start

```bash
# Buka langsung di browser
open index.html

# Atau gunakan local server
python -m http.server 8000
# atau
npx serve .
```

## 📁 Struktur Project

```
sdit-cahayailmu/
├── index.html                 # Halaman utama
├── styles.css                 # Stylesheet utama
├── script.js                  # JavaScript functionality
├── assets/                    # Assets website
│   └── img/                   # Gambar-gambar website

└── README.md                  # Dokumentasi ini
```

## 📝 Customization

### Mengubah Warna Tema
Edit CSS variables di `styles.css`:
```css
:root {
    --primary-color: #667eea;
    --secondary-color: #764ba2;
    --accent-color: #ffd700;
}
```

### Mengubah Font
Ganti font family di CSS:
```css
body {
    font-family: 'Your-Font', sans-serif;
}
```

### Menambahkan Section Baru
1. Tambahkan HTML section di `index.html`
2. Tambahkan styling di `styles.css`
3. Tambahkan JavaScript functionality jika diperlukan
